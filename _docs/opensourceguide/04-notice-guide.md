# NOTICE 가이드

오픈소스를 재배포하는 경우 기본적으로 NOTICE라는 파일에 저작권과 라이선스 정보를 포함한 내용을 고지할 의무가 있습니다. 

이에 포함될 내용은 다음과 같은 것들이 있습니다.
- 사용된 오픈소스의 명칭
- 오픈소스에 접근할 수 있는 URL
- 라이선스 원문 (보통 하이퍼링크 정보)
- 저작권 
- 소스 코드를 받는 방법 (GPL과 같이 소스코드 공개 의무가 발생하는 경우)
